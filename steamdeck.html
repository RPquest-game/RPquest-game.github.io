Play RPquest on your Steam Deck


RPquest is perfectly compatible with your Steam Deck console if you left it running SteamOS. If you use your Steam Deck with a keyboard, it is best to install the basic Linux version. Playing on Steam Deck gives you early access to certain game updates. Attention ! please run the game only in desktop mode and never in gaming mode, where it will not work! Please have not changed the controller desktop configuration for gaming.
